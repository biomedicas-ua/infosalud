Compromiso con la Accesibilidad

Estamos comprometidos a garantizar que nuestro sitio web sea accesible para todas las personas, incluidas aquellas con discapacidades. Esta declaración describe cómo hemos abordado la accesibilidad en nuestro contenido web y qué medidas hemos implementado.

Identificación del Idioma Principal

El idioma principal de este sitio web está definido como español. Se ha implementado el atributo lang="es" en la etiqueta html para asegurar que las tecnologías de asistencia puedan identificar correctamente el idioma del contenido.

Etiquetado Semántico

Hemos utilizado un etiquetado semántico adecuado en todas las páginas de nuestro sitio. Esto incluye el uso correcto de etiquetas como h1, h2, p, y listas (ul y ol) para estructurar el contenido de manera que sea comprensible para los usuarios y las tecnologías de asistencia.

Texto Alternativo de las Imágenes

Todas las imágenes que aparecen en este sitio cuentan con un texto alternativo que describe su contenido o función. Se ha utilizado el atributo alt en todas las imágenes relevantes. Las imágenes decorativas tienen el atributo alt="" para que no se lean por tecnologías de asistencia.

Mecanismos de Navegación Implementados

Hemos implementado varios mecanismos de navegación que facilitan el acceso al contenido:

  • Menús de navegación claros y consistentes que se mantienen en todas las páginas.
  • Enlaces que indican claramente su propósito, evitando frases vagas como "leer más".
  • Uso de encabezados estructurados para facilitar la navegación con lectores de pantalla.

Características de Accesibilidad Implementadas

  • Contraste de Color: Se ha asegurado que los colores del texto y del fondo cumplan con los estándares de contraste para garantizar la legibilidad.
  • Contenido Multimedia: Todos los videos tienen subtítulos y descripciones, cuando es necesario, para garantizar que el contenido multimedia sea accesible.
  • Accesibilidad con Teclado: Todas las funciones del sitio son accesibles mediante el teclado, permitiendo a los usuarios navegar sin necesidad de un ratón.
  • Diseño:está diseñado para su correcta visualización en dispositivos de escritorio, tablet y móviles.

Estilos alternativos

En nuestro compromiso de hacer que este sitio web sea accesible para todos los usuarios, hemos implementado una serie de estilos alternativos que permiten a los visitantes personalizar la apariencia de la página según sus necesidades o preferencias. Estos estilos están diseñados para mejorar la legibilidad y reducir la fatiga visual, especialmente para personas con discapacidades visuales o dificultades de lectura. A continuación, describimos los diferentes estilos que están disponibles:

  1. Modo Noche:El Modo Noche ofrece un esquema de colores diseñado para entornos con poca luz. El fondo oscuro y el texto claro ayudan a reducir el deslumbramiento y la fatiga ocular, lo que facilita la lectura en situaciones de poca luz. Este modo es ideal para usuarios que prefieren un contraste suave en lugar de la tradicional combinación de fondo claro y texto oscuro.
  2. Bajo contraste:El estilo de Bajo Contraste reduce el contraste entre el texto y el fondo, haciendo que la combinación de colores sea más suave para los ojos. Esto es especialmente útil para personas con hipersensibilidad al contraste elevado, lo que puede causar fatiga visual o dificultades para leer texto durante períodos prolongados.
  3. Letra mayor:El estilo de Letra Mayor aumenta significativamente el tamaño del texto en toda la página, mejorando la legibilidad para usuarios con baja visión o dificultades para leer fuentes pequeñas. Esto permite a los usuarios navegar por el contenido de manera más cómoda, sin necesidad de realizar zoom manual en cada página.
  4. Bajo contraste y letra mayor:Este estilo combina las características de Bajo Contraste y Letra Mayor para proporcionar una experiencia de lectura más suave y accesible. Con este modo, los usuarios se benefician de una reducción del contraste visual, mientras que el texto de mayor tamaño facilita aún más la lectura. Este estilo es particularmente útil para personas con baja visión o sensibilidad al brillo y contraste elevado.

Estilos de impresion

ste sitio web incluye un estilo de impresión accesible que optimiza el contenido para la salida en papel. El diseño elimina elementos visuales innecesarios, como imágenes decorativas y menús, y ajusta el formato del texto para garantizar una lectura clara y fluida en formato impreso. Este estilo garantiza que la información esencial se presente de manera clara y sin distracciones, mejorando la accesibilidad del contenido cuando se imprime.

Estado de Conformidad

Este sitio web es parcialmente conforme con el nivel AA de las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1.
